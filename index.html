عن هذا المثال

تم إنشاء هذا الموقع كنموذج تعليمي يوضح كيفية بناء صفحة هبوط بسيطة تدعم عدة لغات باستخدام HTML وCSS وJavaScript فقط، مع إمكانية رفعها على GitHub Pages.

يمكنك استخدام هذا الكود كنقطة بداية لموقع شخصي، صفحة مشروع، أو حتى صفحة تعريف لخدمة تقدمها. كل ما عليك هو تعديل النصوص والألوان بما يناسبك.

في ملف JavaScript ستجد كائن الترجمة الذي يحتوي على النصوص لكل لغة. يمكنك إضافة لغات أخرى أو تعديل النصوص الحالية بسهولة.

معلومات تقنية

  • الهيكل: HTML5 مع تقسيم واضح للأقسام.
  • التصميم: CSS مع Grid وFlexbox.
  • التفاعل: JavaScript بسيط لتبديل اللغة.
  • الاستضافة: GitHub Pages مجاناً.
